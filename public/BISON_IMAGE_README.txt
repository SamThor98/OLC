BISON IMAGE SETUP
=================

To use your own bison image, place a file named "bison-storm.jpg" in this public folder.

The image should show a bison heading through a storm, representing perseverance.

If the image is not found, the website will automatically use a fallback image from Unsplash.

Recommended image specifications:
- Format: JPG or PNG
- Size: At least 400x400 pixels (larger is better for quality)
- Aspect ratio: Square (1:1) works best for the circular displays
- File name: bison-storm.jpg

The image will be displayed in:
- Header logo (small, 48x48px)
- Hero section (large, 128-160px)
- About section (medium, 96-128px)
- Footer (small, 80x80px)
